# 苔原小勇者 · Mossbound Journey

一个无需安装、可直接在浏览器游玩的卡通横版闯关游戏。

## 在线游玩

https://upwardand.github.io/mossbound-journey/

## 操作方式

- `A` / `D` 或方向键：移动
- `W` / 空格键：跳跃，长按可以跳得更高
- `Shift` / `J`：冲刺，可击破普通敌人
- `L`：在标题页打开已解锁关卡选择
- `P`：暂停
- `R`：重开当前关

游戏包含移动平台、碎裂平台、弹跳台、四类敌人和三场首领战，并带有动态背景音乐、连击计分、通关评价和已解锁关卡选择。手机和平板会自动显示移动、冲刺和跳跃按钮。游戏进度保存在玩家自己的浏览器中。
